## Mi Prompt Navegación 3 Páginas 

Lenguaje dart, flutter. Nivel principiante. Ejemplo de navegacion entre 3 paginas utilizando rutas nombradas, desde main llamar a pagina 1.  En la pagina uno mostrar en appbar el titulo "Inicio Parisina", el appbar tendra color de texto blanco y fondo Rojo Oscuro  y en body en una columna mostrar texto (Pagina inicial) mas un boton para ir a la segunda pagina, en la segunda pagina mostrar el appbar "Segunda pagina" color de texto blanco y fondo Rojo Oscuro y en body mostrar una imagen desde la red mas un boton para ir a la tercera pagina, en la tercera pagina en appbar titulo "Tercera pagina" color de texto blanco y fondo Rojo Oscuro y en body mostrar un container de 200 x 200 color crema. Debe ser atractivo facil navegacion y mostrar todo el codigo en un solo archivo.
